# EduCenter OS

EduCenter OS is a cloud-native coaching center management platform.

The goal of this project is to build a real-world full-stack application and deploy it using enterprise-style DevOps practices.

## V1 Scope

Version 1 will support only Admin user.

Admin can:

- Login
- View dashboard
- Manage students
- Manage courses
- Manage batches

## Tech Stack

- Frontend: Next.js + Tailwind CSS
- Backend: FastAPI
- Database: PostgreSQL
- Local Development: Docker Compose
- Future Deployment: K3s Kubernetes
- Future CI/CD: Jenkins
- Future GitOps: Argo CD
